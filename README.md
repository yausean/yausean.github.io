# yausean.github.io

A Jekyll-powered website for GitHub Pages.

## Run locally

1. Install gems:
   `bundle install --path vendor/bundle`
2. Start the site:
   `bundle exec jekyll serve`
3. Open:
   `http://127.0.0.1:4000`

## Customize

- Edit `_config.yml` for site-wide settings
- Update `index.md` and `about.md` for content
- Adjust `assets/css/style.css` for the visual design
